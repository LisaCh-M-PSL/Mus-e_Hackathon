Notre but est de créer un audioguide pour le musée des Mines de Paris.

A partir de rien, nous avons créé un site web adapté pour un format téléphone - format le plus pratique pour une visite dans un musée -. 

Nous avons commencé par identifier les attendus et nous avons ainsi divisé notre site web en 5 parties :
 - Une page d'accueil
 - Une page avec un plan interactif qui permet de cliquer sur les salles concernées et afficher les descriptions des minéraux
 - Une page permettant d'aller directement sur les salles
 - Une page QR CODE qui permet de scanner ceux de la salle 
 - Une page informations qui contientles horaires d'ouverture du musée, les tarifs, les accès, les liens vers leurs sites et réseaux sociaux, et le choix des langues
